# 4. 시연 시나리오

## 회원가입 페이지

1. 사용자는 가입되지 않은 이메일을 작성한 후 인증번호 요청을 한다.
2. 해당 이메일로 전송받은 인증번호를 입력한 후 확인을 클릭한다.(스팸 메일함 확인)
3. 비밀번호, 비밀번호 확인을 조건에 맞추어 입력한다.

- 8자~16자, 영문 소문자 최소 1자, 숫자 최소 1자, 특수문자(~!@#$%^&\*()) 중 최소 1자

4. 닉네임 입력 후 중복 확인을 진행한다.

- 닉네임은 영문, 숫자, (\_)만 입력이 가능하며 5~10자로 구성해야 한다.

5. 생년월일과 전화번호 8자리를 입력한다.
6. 인증 완료된 후 이메일 필드를 조작할 경우 이메일 재인증이 필요함에 유의해야 한다.
7. 모든 조건을 충족시켰으면 회원가입에 성공하고 추가 정보 입력 페이지로 이동한다.

## 추가 정보 입력 페이지

1. 사용자는 성별, 키, 몸무게, 지역의 추가 정보를 입력할 수 있다.
2. 원하지 않는 경우 다음에 입력할 수 있다.
3. 입력을 완료했거나, 다음에 입력하기를 누른 경우 로그인 페이지로 이동한다.

## 로그인 페이지

1. 사용자는 로그인 페이지에서 이메일, 비밀번호를 치고 로그인을 한다.

- 계정이 없는 경우 중앙의 로그인 버튼 하단의 회원가입을 진행한다.
- 이메일, 비밀번호를 잃어버린 경우 이메일 찾기, 비밀번호 찾기로 이동할 수 있다.
- 소셜 로그인을 진행하고자 하는 경우 원하는 소셜 도메인에 맞는 소셜 로그인을 진행할 수 있다.

2. 잘못된 입력을 한 경우(email, 비밀번호 오기입) 로그인이 되지 않는다.
3. 로그인 성공 시 메인 페이지로 이동한다.

## 이메일 찾기 페이지

1. 사용자는 이메일 찾기 페이지에서 이름, 전화번호를 입력한다.
2. 사용자는 본인이 입력한 정보와 일치하는 이메일을 반환 받는다.
3. 일치하는 정보가 없는 경우 일치하는 이메일이 없다는 알림창이 뜬다.
4. 사용자는 이메일을 반환 받고 로그인 페이지로 이동할지 비밀번호 찾기 페이지로 이동할지 선택할 수 있다.

## 비밀번호 찾기 페이지

1. 사용자는 이름, 전화번호, 본인이 가입한 이메일을 작성하고 인증을 진행한다.
2. 인증이 완료된 경우 비밀번호 변경 페이지로 이동한다.

## 비밀번호 변경 페이지

1. 비밀번호를 조건에 맞게 입력한 후 비밀번호 변경을 누르면 새 비밀번호로 변경된다.

## 메인 페이지

1. 메인 페이지 상단 내비게이션 바에는 현위치, 알림, 프로필 이미지가 나타난다.

- 프로필 이미지를 누를 경우 프로필 페이지로 이동한다.
- 알림 아이콘을 누를 경우 알람 내역으로 이동한다.

2. 중상단에는 좌우스크롤이 가능한 추천 루틴이 제공된다.

- 각 루틴을 클릭할 경우 각 루틴의 상세 정보 페이지로 이동한다.

3. 중단 이하로는 내 최근 루틴이 표시되며, 즐겨찾기한 루틴의 경우 상단에 위치하게 된다.

- 중단의 우상단의 더보기를 누르면 전체 루틴 목록이 보여진다.
- 각 루틴의 상세보기를 누르면 각 루틴이 나타난다

4. 하단의 내비게이션 바를 통해 각 위치로 이동할 수 있다.
5. 현재 진행 중인 운동이 존재하는 경우 중단의 진행중인 운동 버튼을 클릭해 진행 중인 운동으로 이동할 수 있다.

- 존재하지 않는 경우, 진행 중인 운동이 없다는 안내창이 뜬다.

## 운동 페이지

1. 운동 목록에서는 부위별 운동들을 확인할 수 있다. 각 운동에 대해 찜을 할 수 있고, 찜한 운동은 목록 상단에 위치한다.
2. 사용자는 각 탭을 클릭하여 각 부위의 운동을 확인할 수 있고, 상단의 검색창을 이용해 검색할 수 있다.
3. 각 운동에서 체크박스를 클릭하면 하단에 운동 추가하기 버튼이 생성된다.

- 운동 추가하기 버튼을 누르면 루틴 생성 페이지로 이동한다.

4. 사용자는 각 운동을 클릭해 운동 상세 정보를 확인할 수 있다.

- 상세정보에서는 큰 운동 예시 화면과 설명이 나타나며, 이 페이지에서도 우상단의 찜, 운동 추가 기능을 이용할 수 있다.

## 기록 페이지

1. 초기 화면은 달력이 표시된다. 달력을 조작하여 원하는 달로 이동할 수 있다.
2. 각 달력에서 초록색 글씨는 루틴이 존재하는 날을, 빨간 글씨는 선택된 날을, 하얀 글씨는 루틴이 없는 날을 의미한다.
3. 루틴이 생성된 날은 중단의 루틴 상세보기 버튼으로 해당 페이지로 이동할 수 있다.
4. 루틴이 생성되지 않은 날은 루틴 생성하기 버튼으로 루틴 생성을 진행할 수 있다.
5. 하단의 인바디 등록, 결과, 운동 기록 탭을 통해 각 페이지로 이동할 수 있다.

## 루틴&운동 페이지

1. 사용자는 루틴 생성 페이지에 들어와서 루틴 생성을 진행할 수 있다. 상단에서 루틴 제목과 실행 날짜를 입력하고 운동 추가 버튼을 통해 운동 목록에서 운동을 추가할 수 있다.
2. 운동이 추가된 경우 세트 추가/삭제를 통해 하고자 하는 운동의 무게 횟수 등을 조절할 수 있다.
3. 오늘자 루틴을 생성한 경우 우하단의 운동 시작하기 버튼을 통해 운동을 시작할 수 있다.
4. 운동을 시작하면 좌하단의 타이머가 동작한다.
5. 사용자는 각 세트의 체크박스를 조작하여 운동 완료 여부를 표시할 수 있으며, 각 세트 완료 후 1분의 휴식 시간이 부여된다.
6. 사용자는 중간에 우 하단의 운동 종료 버튼을 클릭해 운동을 종료할 수 있다. 한 번 종료된 운동은 재시작할 수 없다.
7. 운동이 종료된 경우 운동 결과 페이지가 나타나며, 총 운동량, 운동 시간이 나타난다. 우하단의 업로드 버튼을 통해 피드 업로드 페이지로 이동할 수 있다.

## 인바디 페이지

1. 사용자는 촬영 혹은 갤러리 버튼을 이용하여 인바디 결과지를 등록할 수 있다.
2. 인바디 결과지를 인식해 폼으로 이동하며, 사용자가 인식 결과를 수정/등록할 수 있는 폼을 제공한다.
3. 인바디 결과 페이지에서는 사용자가 등록한 인바디 이력을 그래프를 통해 확인할 수 있다.

- 유사한 분류별로 묶여서 제공해준다.
- 사용자는 각 탭을 클릭하여 원하는 정보 이력을 확인할 수 있다.

## 운동 기록 페이지

1. 사용자는 운동 기록 페이지에서 자신의 최근 운동 이력을 확인할 수 있다.
2. 확인 가능한 운동 이력은 상단부터 3대 운동의 1RM 변화 추이, 총 운동량, 운동시간 변화 추이를 확인할 수 있다.

## AI 트레이닝 페이지

1. 사용자는 초기 설정 모달을 이용해 원하는 운동과 원하는 횟수를 선택할 수 있다.
2. 초기 설정이 완료된 경우 운동 준비 -> 카메라 시작 버튼을 누르면 5초의 준비 시간이 부여되고 카메라가 시작되며 운동을 감지한다.
3. 올바른 자세를 취했을 경우 O, 올바르지 않은 자세인 경우 X가 표시된다.
4. 지정한 횟수가 모두 카운트 된 경우 결과 모달 창의 결과와 코멘트를 통해 본인의 운동 자세에 대한 코멘트를 확인할 수 있다.

## SNS 페이지

### 프로필

1. 상단에는 공통적으로 프로필 이미지, 닉네임, 운동 피드 수, 거래 게시글 수, 팔로잉, 팔로워 수가 나타난다.

- 자신의 프로필에 접속한 경우 상단의 두 버튼이 채팅목록, 내 정보 버튼으로 나타난다. 내 정보를 클릭하는 경우 내 정보 페이지로 이동한다.
- 상대방의 프로필에 접속한 경우 상단의 두 버튼이 채팅하기, 팔로우/취소 버튼으로 나타난다.

2. 중단의 운동/거래 탭을 조작하여 각 분류의 게시글을 확인할 수 있다.
3. 최상단의 검색창을 이용해 유저들의 닉네임을 검색할 수 있다.

### 피드

1. 피드는 자신의 팔로우한 유저의 피드들이 나타난다. 각 피드에 대해 좋아요, 댓글이 가능하다.
2. 피드 작성은 당일 운동을 완료한 경우에만 가능하며 과거의 운동 피드를 올릴 수는 없다.
3. 피드에는 이미지와 내용(필수 아님)을 작성할 수 있다.
4. 루틴 보기 버튼을 이용해 루틴을 확인할 수 있다.

### 장터

1. 장터에서는 사용자들이 올린 게시글을 확인할 수 있다.
2. 중상단의 드롭다운과 검색창을 이용해 광역자치단체, 기초자치단체, 검색어 별 필터링이 가능하다.
3. 각 게시글을 클릭하면 게시글 모달이 나타난다.
4. 각 게시글 모달에서는 판매자의 정보와 판매 상태를 확인할 수 있고, 판매자와 채팅을 진행할 수 있다. 만약 사용자가 판매자인 경우 채팅 목록으로 이동하게 된다.
5. 게시글은 이미지 1장 이상, 제목, 가격, 내용을 모두 적어야 게시가 가능하다.(이미지 최대 5장)

## 채팅

1. 사용자는 유저 프로필, 판매 게시글을 통해 각 유저와의 채팅 페이지에 진입할 수 있다.
2. 잠시 동안의 로딩 후 채팅 페이지가 활성화되면 이전 채팅 정보가 불러와지며 해당 유저와 채팅을 진행할 수 있다.
3. 이전 채팅을 보기 위해 스크롤 했더라도, 채팅을 전송하면 바로 하단으로 이동한다.
